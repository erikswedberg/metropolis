metropolis

metropolis is a project by Robert Harris, Erik Swedberg, and Nathan Chan.

It was submitted as part of the datacanvas.org Sense Your City Data Art Visualisation Challenge. The code is available at github here.

It makes use of the following technologies

  • d3.js
  • Python, Flask, PostgreSQL
  • R

The project is intended to convey a sense of data changing over time. Each data point is one city for one sensor measurement for one hour, and a block represents a whole week. The colors are chosen based on a linear interpolation of the range of all the data for that sensor type.
